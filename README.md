# PCAP-SEM2-GS-LEE

Este é um sistema simples desenvolvido em Python orientado a objetos que simula uma ferramenta inteligente de orientação de carreiras, focando em organizar e analisar perfis profissionais do futuro. Ele conecta lógica de programação com desenvolvimento humano, permitindo cadastrar perfis com competências técnicas e comportamentais armazenadas em dicionários, como lógica, criatividade, colaboração e adaptabilidade. Com base nessas informações, o sistema gera recomendações personalizadas de carreiras, trilhas de aprendizado ou áreas de aprimoramento, preparando usuários para o trabalho do futuro e incentivando aprendizado contínuo.

As funcionalidades incluem cadastro de perfis com nome e competências (ex: lógica nível 8), análise que compara competências com requisitos de carreiras pré-definidas para sugerir opções compatíveis (pelo menos 70% de match), armazenamento simples via arquivos JSON simulando um banco de dados, interface textual (CLI) com menu para interação, e extensibilidade para adicionar novas carreiras ou competências editando JSONs. As competências são flexíveis, armazenadas como chaves em dicionários (ex: {"logica": 8}), e você pode usar qualquer string como nome, sem lista fixa. Exemplos pré-definidos incluem lógica (raciocínio lógico), criatividade (inovação), colaboração (trabalho em equipe) e adaptabilidade (flexibilidade), mas você pode adicionar tipos como liderança ou comunicação ao cadastrar perfis ou editar carreiras.json.

O projeto é organizado em módulos: classes.py define as classes Competencia, Perfil e Carreira; main.py contém funções para salvar/carregar JSON, gerar recomendações e o menu CLI. Os arquivos carreiras.json (com carreiras pré-definidas) e perfis.json (inicialmente vazio, atualizado ao cadastrar) devem estar no mesmo diretório. O sistema cria JSONs automaticamente se não existirem.

Um exemplo de uso: cadastre um perfil como "João" com "logica 8" e "criatividade 7", então peça recomendações para ver sugestões como "Desenvolvedor de Software".
